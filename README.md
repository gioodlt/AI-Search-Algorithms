# AI-Search-Algorithms
Implementation of classical AI search algorithms in Python, including BFS, DFS, Uniform Cost Search, and A* Search.
AI Search Algorithms


## Overview

This project implements four fundamental artificial intelligence search algorithms used for graph traversal and pathfinding.

## Algorithms implemented:

- Breadth-First Search (BFS)
- Depth-First Search (DFS)
- Uniform Cost Search (UCS)
- A* Search

## Technologies

- Python
- Object-Oriented Programming
- Graph Algorithms
- Priority Queues
